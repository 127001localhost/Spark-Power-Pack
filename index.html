Spark Power Pack

Using the Spark APIs, we have created a collection of tools to enable users to perform repetitive tasks in bulk.

 
Invites

invite 100s of people
to a Spark room
Broadcasts

send a message to multiple
Spark rooms
Exodus

manage your Spark Room
memberships and the
users in them
Jabber Alerts

receive Spark messages
in Jabber


In order to start using the Spark Power Pack, you must grant the application access to your Spark account.
